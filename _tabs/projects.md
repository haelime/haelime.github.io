## Projects

### Low-Level Systems

C++ 기반 제어 로직, 프로세스 감시, 데이터 정합성 보장에 관심을 두고 있습니다. 자율주행 셔틀 개발 경험을 바탕으로 FSM, watchdog, fail-safe 구조를 실제 시스템에 적용했습니다.

### Graphics and Simulation

DOTS와 Compute Shader를 활용한 대규모 시뮬레이션 환경 구축에 집중하고 있습니다. 많은 객체와 복잡한 연산 부하를 통제하기 위해 데이터 배치, 병렬 처리, GPU 연산 흐름을 함께 설계합니다.

### Networking

커스텀 바이너리 패킷 통신과 권한형 서버 시뮬레이션을 직접 설계하며 네트워크 트래픽을 줄이고 데이터 정합성을 보장하는 구조를 실험하고 있습니다.
