## Profile

C++와 그래픽스 렌더링을 중심으로 게임 개발을 공부하고 있습니다. 직접 엔진과 렌더러를 만들면서, API 사용법보다 프레임이 어떤 데이터 흐름으로 완성되는지 이해하는 데 관심이 많습니다.
